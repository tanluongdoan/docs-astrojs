> **AstroResponseHeadersReassigned**: Se pueden añadir y eliminar cabeceras individuales de `Astro.response.headers`, pero no se debe sustituir por otra instancia de `Headers`.

## ¿Qué salió mal?
Se produce cuando se establece un valor como campo `headers` en el objeto `ResponseInit` disponible como `Astro.response`.
