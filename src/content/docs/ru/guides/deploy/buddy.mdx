Вы можете развернуть свой проект Astro с помощью [Buddy](https://buddy.works/), CI/CD-решения, которое может собрать ваш сайт и отправить его на различные целевые платформы, включая FTP-серверы и облачные хостинг-провайдеры.

:::note
Buddy сам по себе не будет размещать ваш сайт. Вместо этого он помогает вам управлять процессом сборки и доставлять результат на выбранную вами платформу для развертывания.
:::

## Как развернуть

1. [Создайте учетную запись **Buddy**](https://buddy.works/sign-up).
2. Создайте новый проект и свяжите его с git-репозиторием (GitHub, GitLab, BitBucket, любой частный Git-репозиторий или вы можете использовать собственный хостинг Git от Buddy).
3. Добавьте новый конвейер.
4. Во вновь созданный конвейер добавьте действие **[Node.js](https://buddy.works/actions/node-js)**.
5. В этом действии добавьте:

   ```bash
   npm install
   npm run build
   ```

6. Добавьте действие развертывания — их много на выбор, вы можете просмотреть их в [каталоге действий Buddy](https://buddy.works/actions). Хотя их настройки могут отличаться, не забудьте установить **Source path** в `dist`.
7. Нажмите кнопку **Run**.
