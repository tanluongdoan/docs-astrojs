Вы можете развернуть свой проект Astro на собственном виртуальном частном сервере (VPS) с помощью [Cleavr](https://cleavr.io/), инструмента управления развертыванием серверов и приложений.

:::tip
Ознакомьтесь с [руководством по Astro в документации Cleavr](https://docs.cleavr.io/guides/astro)!
:::

## Необходимые условия 

Чтобы начать работу, вам понадобятся:

- Учетная запись Cleavr
- Сервер на VPS провайдера, использующего Cleavr.

## Добавьте свой сайт

1. В Cleavr перейдите на сервер, к которому вы хотите добавить ваш проект Astro.

2. Выберите **Add Site** и заполните данные о вашем приложении, например доменное имя.

3. Для **App Type** выберите 'NodeJS Static' или 'NodeJS SSR' в зависимости от того, как вы настраиваете свое приложение Astro.
   
4. Для приложений Static установите **Artifact Folder** на `dist`.
   
5. Для приложений SSR:
   - Установите **Entry Point** на `entry.mjs`.
   - Установите **Artifact Folder** на `dist/server`.
   
6. Выберите **Add**, чтобы добавить сайт на ваш сервер.


## Настройка и развертывание

1. После того как новый сайт добавлен, нажмите **Setup and deploy**.
   
2. Выберите **VC Profile**, **Repo** и **Branch** для вашего проекта Astro.
   
3. Внесите любые дополнительные настройки, необходимые для вашего проекта.
   
4. Перейдите на вкладку **Deployments**, а затем нажмите **Deploy**.

Поздравляем, вы только что развернули ваше приложение Astro!
