Вы можете развернуть свой проект Astro на [Edgio](https://www.edg.io/), платформе для развертывания, защиты и ускорения веб-сайтов и API с кэшированием на краю сети (edge).

:::tip
Ознакомьтесь с [руководством по Astro в документации Edgio](https://docs.edg.io/guides/astro)!
:::

## Как развернуть

1. Установите [Edgio CLI](https://docs.edg.io/guides/cli) глобально из терминала, если вы этого еще не сделали.

    ```bash
    npm install -g @edgio/cli
    ```

2. Добавьте Edgio на ваш сайт Astro

    ```bash
    edgio init
    ```

3. (Необязательно) Включите рендеринг на стороне сервера

    После настройки [@astrojs/node с Astro](/ru/guides/integrations-guide/node/), укажите путь к файлу сервера в `edgio.config.js`, как показано ниже:

    ```js ins={2,5-9}
    // edgio.config.js
    import { join } from 'path'

    module.exports = {
      astro: {
        // Путь к автономному серверу, на котором работает Astro SSR.
        // Зависимости для этого файла собираются автоматически.
        appPath: join(process.cwd(), 'dist', 'server', 'entry.mjs'),
      },
    };
    ```

4. Разверните на Edgio

    ```bash
    edgio deploy
    ```
