Вы можете развернуть сайт Astro с помощью [Flightcontrol](https://www.flightcontrol.dev?ref=astro), который обеспечивает полностью автоматизированное развертывание на вашем аккаунте AWS.

Поддерживаются как статические, так и SSR-сайты Astro.

## Как развернуть

1. Создайте учетную запись Flightcontrol на [app.flightcontrol.dev/signup](https://app.flightcontrol.dev/signup?ref=astro).
2. Перейдите по адресу [app.flightcontrol.dev/projects/new/1](https://app.flightcontrol.dev/projects/new/1)
3. Подключите свой аккаунт на GitHub и выберите репозиторий.
4. Выберите желаемый "Тип конфигурации":
    - `GUI` (вся конфигурация управляется через панель Flightcontrol), где вы выберете предустановленный набор `Astro Static` или `Astro SSR`.
    - `flightcontrol.json` (вариант "инфраструктура как код", когда весь конфиг находится в вашем репозитории), где вы выберете пример конфига Astro, а затем добавите его в вашу кодовую базу как `flightcontrol.json`.
5. Внесите необходимые изменения в конфигурацию при необходимости
6. Нажмите "Create Project" и выполните все необходимые шаги (например, привязку вашего аккаунта AWS).


### Настройка SSR

Чтобы развернуть проект с поддержкой SSR, сначала настройте адаптер [`@astrojs/node`](/ru/guides/integrations-guide/node/). Затем выполните описанные выше шаги, выбрав соответствующие конфигурации для Astro SSR.
