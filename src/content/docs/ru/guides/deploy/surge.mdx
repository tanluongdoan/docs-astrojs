Вы можете развернуть свой проект Astro на [Surge](https://surge.sh/), платформе веб-публикации с одной командой, предназначенной для разработчиков фронтенда.

## Как развернуть

1. Установите [Surge CLI](https://www.npmjs.com/package/surge) глобально из Терминала, если вы этого еще не сделали.

    ```shell
    npm install -g surge
    ```

2. Создайте свой сайт Astro из корневой директории проекта.

    ```shell
    npm run build
    ```

3. Разверните сайт на Surge с помощью CLI.

    ```shell
    surge dist
    ```

    Вы можете использовать [пользовательский домен](http://surge.sh/help/adding-a-custom-domain) при развертывании, выполнив команду `surge dist yourdomain.com`.
