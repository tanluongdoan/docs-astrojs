Ниже приведен полный список ошибок, с которыми вы можете столкнуться при работе с Astro. Для получения дополнительной помощи, в том числе по распространенным ошибкам, обратитесь к нашему [Руководству по устранению неполадок](/ru/guides/troubleshooting/).

## Ошибки Astro

- [**UnknownCompilerError**](/ru/reference/errors/unknown-compiler-error/)<br/>Unknown compiler error.
- [**StaticRedirectNotAvailable**](/ru/reference/errors/static-redirect-not-available/)<br/>`Astro.redirect` is not available in static mode.
- [**ClientAddressNotAvailable**](/ru/reference/errors/client-address-not-available/)<br/>`Astro.clientAddress` is not available in current adapter.
- [**StaticClientAddressNotAvailable**](/ru/reference/errors/static-client-address-not-available/)<br/>`Astro.clientAddress` is not available in static mode.
- [**NoMatchingStaticPathFound**](/ru/reference/errors/no-matching-static-path-found/)<br/>No static path found for requested path.
- [**OnlyResponseCanBeReturned**](/ru/reference/errors/only-response-can-be-returned/)<br/>Invalid type returned by Astro page.
- [**MissingMediaQueryDirective**](/ru/reference/errors/missing-media-query-directive/)<br/>Missing value for `client:media` directive.
- [**NoMatchingRenderer**](/ru/reference/errors/no-matching-renderer/)<br/>No matching renderer found.
- [**NoClientEntrypoint**](/ru/reference/errors/no-client-entrypoint/)<br/>No client entrypoint specified in renderer.
- [**NoClientOnlyHint**](/ru/reference/errors/no-client-only-hint/)<br/>Missing hint on `client:only` directive.
- [**InvalidGetStaticPathParam**](/ru/reference/errors/invalid-get-static-path-param/)<br/>Invalid value returned by a `getStaticPaths` path.
- [**InvalidGetStaticPathsEntry**](/ru/reference/errors/invalid-get-static-paths-entry/)<br/>Invalid entry inside getStaticPath's return value
- [**InvalidGetStaticPathsReturn**](/ru/reference/errors/invalid-get-static-paths-return/)<br/>Invalid value returned by getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/ru/reference/errors/get-static-paths-removed-rsshelper/)<br/>getStaticPaths RSS helper is not available anymore.
- [**GetStaticPathsExpectedParams**](/ru/reference/errors/get-static-paths-expected-params/)<br/>Missing params property on `getStaticPaths` route.
- [**GetStaticPathsInvalidRouteParam**](/ru/reference/errors/get-static-paths-invalid-route-param/)<br/>Invalid value for `getStaticPaths` route parameter.
- [**GetStaticPathsRequired**](/ru/reference/errors/get-static-paths-required/)<br/>`getStaticPaths()` function required for dynamic routes.
- [**ReservedSlotName**](/ru/reference/errors/reserved-slot-name/)<br/>Invalid slot name.
- [**NoAdapterInstalled**](/ru/reference/errors/no-adapter-installed/)<br/>Cannot use Server-side Rendering without an adapter.
- [**NoMatchingImport**](/ru/reference/errors/no-matching-import/)<br/>No import found for component.
- [**InvalidPrerenderExport**](/ru/reference/errors/invalid-prerender-export/)<br/>Invalid prerender export.
- [**InvalidComponentArgs**](/ru/reference/errors/invalid-component-args/)<br/>Invalid component arguments.
- [**PageNumberParamNotFound**](/ru/reference/errors/page-number-param-not-found/)<br/>Page number param not found.
- [**ImageMissingAlt**](/ru/reference/errors/image-missing-alt/)<br/>Image missing required "alt" property.
- [**InvalidImageService**](/ru/reference/errors/invalid-image-service/)<br/>Error while loading image service.
- [**MissingImageDimension**](/ru/reference/errors/missing-image-dimension/)<br/>Missing image dimensions
- [**FailedToFetchRemoteImageDimensions**](/ru/reference/errors/failed-to-fetch-remote-image-dimensions/)<br/>Failed to retrieve remote image dimensions
- [**UnsupportedImageFormat**](/ru/reference/errors/unsupported-image-format/)<br/>Unsupported image format
- [**UnsupportedImageConversion**](/ru/reference/errors/unsupported-image-conversion/)<br/>Unsupported image conversion
- [**PrerenderDynamicEndpointPathCollide**](/ru/reference/errors/prerender-dynamic-endpoint-path-collide/)<br/>Prerendered dynamic endpoint has path collision.
- [**ExpectedImage**](/ru/reference/errors/expected-image/)<br/>Expected src to be an image.
- [**ExpectedImageOptions**](/ru/reference/errors/expected-image-options/)<br/>Expected image options.
- [**IncompatibleDescriptorOptions**](/ru/reference/errors/incompatible-descriptor-options/)<br/>Cannot set both `densities` and `widths`
- [**ImageNotFound**](/ru/reference/errors/image-not-found/)<br/>Image not found.
- [**NoImageMetadata**](/ru/reference/errors/no-image-metadata/)<br/>Could not process image metadata.
- [**MarkdownImageNotFound**](/ru/reference/errors/markdown-image-not-found/)<br/>Image not found.
- [**CouldNotTransformImage**](/ru/reference/errors/could-not-transform-image/)<br/>Could not transform image.
- [**ResponseSentError**](/ru/reference/errors/response-sent-error/)<br/>Unable to set response.
- [**MiddlewareNoDataOrNextCalled**](/ru/reference/errors/middleware-no-data-or-next-called/)<br/>The middleware didn't return a `Response`.
- [**MiddlewareNotAResponse**](/ru/reference/errors/middleware-not-aresponse/)<br/>The middleware returned something that is not a `Response` object.
- [**EndpointDidNotReturnAResponse**](/ru/reference/errors/endpoint-did-not-return-aresponse/)<br/>The endpoint did not return a `Response`.
- [**LocalsNotAnObject**](/ru/reference/errors/locals-not-an-object/)<br/>Value assigned to `locals` is not accepted.
- [**AstroResponseHeadersReassigned**](/ru/reference/errors/astro-response-headers-reassigned/)<br/>`Astro.response.headers` не должны быть переназначены.
- [**MiddlewareCantBeLoaded**](/ru/reference/errors/middleware-cant-be-loaded/)<br/>Can't load the middleware.
- [**LocalImageUsedWrongly**](/ru/reference/errors/local-image-used-wrongly/)<br/>Local images must be imported.
- [**AstroGlobUsedOutside**](/ru/reference/errors/astro-glob-used-outside/)<br/>Astro.glob() used outside of an Astro file.
- [**AstroGlobNoMatch**](/ru/reference/errors/astro-glob-no-match/)<br/>Astro.glob() did not match any files.
- [**RedirectWithNoLocation**](/ru/reference/errors/redirect-with-no-location/)<br/>A redirect must be given a location with the `Location` header.
- [**InvalidDynamicRoute**](/ru/reference/errors/invalid-dynamic-route/)<br/>Invalid dynamic route.
- [**MissingSharp**](/ru/reference/errors/missing-sharp/)<br/>Could not find Sharp.
- [**UnknownViteError**](/ru/reference/errors/unknown-vite-error/)<br/>Unknown Vite Error.
- [**FailedToLoadModuleSSR**](/ru/reference/errors/failed-to-load-module-ssr/)<br/>Could not import file.
- [**InvalidGlob**](/ru/reference/errors/invalid-glob/)<br/>Invalid glob pattern.
- [**FailedToFindPageMapSSR**](/ru/reference/errors/failed-to-find-page-map-ssr/)<br/>Astro couldn't find the correct page to render
- [**MissingLocale**](/ru/reference/errors/missing-locale/)<br/>The provided locale does not exist.
- [**MissingIndexForInternationalization**](/ru/reference/errors/missing-index-for-internationalization/)<br/>Index page not found.
- [**NoPrerenderedRoutesWithDomains**](/ru/reference/errors/no-prerendered-routes-with-domains/)<br/>Prerendered routes aren't supported when internationalization domains are enabled.
- [**CantRenderPage**](/ru/reference/errors/cant-render-page/)<br/>Astro can't render the route.
- [**UnhandledRejection**](/ru/reference/errors/unhandled-rejection/)<br/>Unhandled rejection
- [**i18nNotEnabled**](/ru/reference/errors/i18n-not-enabled/)<br/>i18n Not Enabled
