:::caution[Устарело]
Несуществующие коллекции больше не приводят к ошибке. Вместо этого выдается предупреждение.
:::

> Коллекция, запрашиваемая с помощью `getCollection()`, не существует.

## Что пошло не так?

При запросе коллекции убедитесь, что каталог коллекции с запрашиваемым именем существует в разделе `src/content/`.
