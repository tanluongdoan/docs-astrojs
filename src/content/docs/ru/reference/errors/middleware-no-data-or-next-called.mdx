> **MiddlewareNoDataOrNextCalled**: Убедитесь, что ваше промежуточное ПО возвращает объект `Response`, либо напрямую, либо возвращая `Response` из вызова функции `next`.

## Что пошло не так?

Возникает, когда промежуточное ПО не возвращает никаких данных или не вызывает функцию `next`.

Например:
```ts
import {defineMiddleware} from "astro:middleware";
export const onRequest = defineMiddleware((context, _) => {
	// не возвращает ничего и не вызывает `next`
	context.locals.someData = false;
});
```
