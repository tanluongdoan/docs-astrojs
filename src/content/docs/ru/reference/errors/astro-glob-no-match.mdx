> **AstroGlobNoMatch**: `Astro.glob(GLOB_STR)` не вернул ни одного подходящего файла.

## Что пошло не так?

`Astro.glob()` не вернул ни одного подходящего файла. Возможно, в шаблоне glob допущена опечатка.
