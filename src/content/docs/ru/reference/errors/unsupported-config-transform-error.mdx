> **UnsupportedConfigTransformError**: Функции `transform()` в вашей конфигурации контента должны возвращать допустимый JSON или типы данных, совместимые с библиотекой devalue (включая Dates, Maps и Sets).<br/>Полная ошибка: PARSE_ERROR	

## Что пошло не так?

Функции `transform()` в вашей конфигурации контента должны возвращать допустимый JSON или типы данных, совместимые с библиотекой devalue (включая Dates, Maps и Sets).	

**Смотрите также:**
-  [Библиотека devalue](https://github.com/rich-harris/devalue)
