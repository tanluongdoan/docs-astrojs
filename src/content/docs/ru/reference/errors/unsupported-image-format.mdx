> **UnsupportedImageFormat**: Получен неподдерживаемый формат `FORMAT` из `IMAGE_PATH`. В настоящее время только форматы SUPPORTED_FORMATS.JOIN(', ') поддерживаются нашими службами обработки изображений.

## Что пошло не так?

Встроенные службы обработки изображений в настоящее время не поддерживают оптимизацию всех форматов изображений.

Для неподдерживаемых форматов, таких как GIF, возможно, вы сможете использовать тег `img` напрямую:
```astro
---
import rocket from '../assets/images/rocket.gif';
---

<img src={rocket.src} width={rocket.width} height={rocket.height} alt="A rocketship in space." />
```
