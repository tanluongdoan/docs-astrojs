> **MissingMediaQueryDirective**: Не предоставлено медиавыражение для директивы `client:media`. Должно быть указано медиавыражение, аналогичное `client:media="(max-width: 600px)"`.

## Что пошло не так?

При использовании директивы `client:media` требуется параметр [медиавыражения](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries).

```astro
<Counter client:media="(max-width: 640px)" />
```
