> **NoClientOnlyHint**: Невозможно отобразить `COMPONENT_NAME`. При использовании стратегии гидратации `client:only`, Astro нуждается в подсказке для использования правильного рендерера.

## Что пошло не так?
Компоненты со стратегией гидратации `client:only` не выполняются на сервере, поэтому Astro не знает (и не может угадать), какой рендерер использовать, и требует подсказку. Например:

```astro
	<SomeReactComponent client:only="react" />
```
