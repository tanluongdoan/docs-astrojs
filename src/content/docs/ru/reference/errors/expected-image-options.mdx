> **ExpectedImageOptions**: Ожидалось, что параметр getImage() будет объектом. Получено `OPTIONS`.

## Что пошло не так?

Первым параметром `getImage()` должен быть объект с различными свойствами, которые нужно применить к изображению.

```ts
import { getImage } from "astro:assets";
import myImage from "../assets/my_image.png";

const optimizedImage = await getImage({src: myImage, width: 300, height: 300});
```

В большинстве случаев эта ошибка возникает из-за того, что параметры были переданы напрямую, а не внутри объекта.
