> **IncompatibleDescriptorOptions**: Можно указать только одно из значений `densities` или `widths`. В большинстве случаев вы, вероятно, захотите использовать только `widths`, если вам нужна определенная ширина.

## Что пошло не так?

Можно указать только один из атрибутов `densities` или `widths`. Эти атрибуты используются для создания атрибута `srcset`, который не может содержать одновременно дескрипторы `x` и `w`.
