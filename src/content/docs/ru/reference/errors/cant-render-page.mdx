> **CantRenderPage**: Astro не может найти контент для рендеринга этого маршрута. С этим маршрутом не связано ни одного файла или перенаправления.

## Что пошло не так?

Astro не смог найти связанный файл с контентом при попытке отображения маршрута. Это ошибка Astro, а не ошибка пользователя. Если перезапуск сервера разработки не устраняет проблему, пожалуйста, создайте запрос.
