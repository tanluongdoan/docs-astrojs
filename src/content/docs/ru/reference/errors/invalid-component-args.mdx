> **Пример сообщения об ошибке:**<br/>
InvalidComponentArgs: Неверные аргументы переданы компоненту `<MyAstroComponent>`.

## Что пошло не так?

Компоненты Astro нельзя отображать вручную с помощью вызова функции, например `Component()` или `{items.map(Component)}`. Предпочтите синтаксис компонента `<Component />` или `{items.map(item => <Component {...item} />)}`.
