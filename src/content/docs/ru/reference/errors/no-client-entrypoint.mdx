> **NoClientEntrypoint**: Компонент `COMPONENT_NAME` имеет директиву `client:CLIENT_DIRECTIVE`, но `RENDERER_NAME` не предоставил клиентскую точку входа для использования при гидратации.

## Что пошло не так?
Astro попытался гидратировать компонент на клиенте, но используемый рендерер не предоставил клиентскую точку входа для использования при гидратации.
