> **MissingIndexForInternationalization**: Не удалось найти индексную страницу. Корневая индексная страница необходима для создания перенаправления на индексный URL локали по умолчанию. (`/DEFAULT_LOCALE`)

## Что пошло не так?

Astro не смог найти главную страницу вашего веб-сайта. Главная страница необходима для того, чтобы Astro мог создать перенаправление с главной страницы на локализованную главную страницу локали по умолчанию при использовании [`i18n.routing.prefixDefaultLocale`](/ru/reference/configuration-reference/#i18nroutingprefixdefaultlocale).
