> **ExpectedImage**: Ожидалось, что свойство `src` для `getImage` или `<Image />` будет либо импортированным ESM изображением, либо строкой с путем к внешнему изображению. Получено значение `SRC` (тип: `TYPEOF_OPTIONS`).<br/><br/>Получены полные сериализованные параметры: `FULL_OPTIONS`."

## Что пошло не так?

Свойство `src` изображения недопустимо. Компонент Image требует, чтобы атрибут `src` был либо импортированным ESM изображением, либо строкой. Это справедливо и для первого параметра функции `getImage()`.

```astro
---
import { Image } from "astro:assets";
import myImage from "../assets/my_image.png";
---

<Image src={myImage} alt="..." />
<Image src="https://example.com/logo.png" width={300} height={300} alt="..." />
```

В большинстве случаев эта ошибка возникает, когда значение, переданное в `src`, не определено.
