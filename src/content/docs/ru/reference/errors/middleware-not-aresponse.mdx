> **MiddlewareNotAResponse**: Любые данные, возвращаемые из промежуточного ПО, должны быть действительным объектом `Response`.

## Что пошло не так?

Возникает в режиме разработки, когда промежуточное ПО возвращает что-то, что не является объектом `Response`.

Например:
```ts
import {defineMiddleware} from "astro:middleware";
export const onRequest = defineMiddleware(() => {
  return "string"
});
```
