> **Пример сообщения об ошибке:**<br/>
Не удалось разобрать frontmatter в **blog** → **post.md**<br/>
"title" обязателен.<br/>
"date" должна быть действительной датой.

## Что пошло не так?

Frontmatter документа Markdown в `src/content/` не соответствуют схеме коллекции.
Убедитесь, что все обязательные поля присутствуют, и что все поля имеют правильный тип.
Вы можете проверить схему коллекции в файле `src/content/config.*`.
Дополнительную информацию можно найти в [документации по коллекциям контента](/ru/guides/content-collections/).
