> **MiddlewareCantBeLoaded**: Произошла неизвестная ошибка при загрузке вашего промежуточного ПО.

## Что пошло не так?

Возникает в режиме разработки, когда промежуточное ПО выдает ошибку при попытке его загрузки.

Например:
```ts
import {defineMiddleware} from "astro:middleware";
throw new Error("Error thrown while loading the middleware.")
export const onRequest = defineMiddleware(() => {
  return "string"
});
```
