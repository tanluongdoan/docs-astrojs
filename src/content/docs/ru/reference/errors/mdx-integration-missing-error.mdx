> **MdxIntegrationMissingError**: Невозможно отобразить FILE. Убедитесь, что установлена интеграция `@astrojs/mdx`.

## Что пошло не так?

Не удалось найти официальную интеграцию `@astrojs/mdx`. Эта ошибка возникает при использовании файлов MDX без установленной интеграции MDX.

**Смотрите также:**

-  [Установка и использование MDX](/ru/guides/integrations-guide/mdx/)
