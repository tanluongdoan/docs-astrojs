> `COLLECTION_NAME` содержит несколько записей с одинаковым слагом: `SLUG`. Слаги должны быть уникальными.

## Что пошло не так?

Записи коллекции контента должны иметь уникальные слаги. Дубликаты часто возникают из-за параметра frontmatter `slug`.
