Эта ошибка больше не является специфической для Markdown и, как таковая, была заменена на `ImageNotFound`
:::

> Не удалось найти запрошенное изображение `IMAGE_PATH` по пути `FULL_IMAGE_PATH`.

## Что пошло не так?

Astro не смог найти изображение, которое вы включили в свой Markdown-контент. Обычно это вызвано простой опечаткой в пути.

Изображения в Markdown относятся к текущему файлу. Чтобы ссылаться на изображение, находящееся в той же папке, что и файл  `.md`, путь должен начинаться с `./`.
