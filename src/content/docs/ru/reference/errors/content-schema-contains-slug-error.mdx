> **ContentSchemaContainsSlugError**: Схема коллекции контента не должна содержать `slug`, поскольку оно зарезервировано для генерации слагов. Удалите это из схемы вашей коллекции COLLECTION_NAME.

## Что пошло не так?

Схема коллекции контента не должна содержать поле `slug`. Оно зарезервировано Astro для генерации слагов записей. Удалите `slug` из вашей схемы. Вы все еще можете использовать пользовательские слаги в frontmatter.
