> `COLLECTION_NAME` → `ENTRY_ID` имеет недопустимый слаг. `slug` должен быть строкой.

## Что пошло не так?

Запись в `src/content/` имеет недопустимый `slug`. Это поле зарезервировано для генерации слагов записей и должно быть строкой, когда присутствует.

**Смотрите также:**
-  [Зарезервированное поле `slug`](/ru/guides/content-collections/)
