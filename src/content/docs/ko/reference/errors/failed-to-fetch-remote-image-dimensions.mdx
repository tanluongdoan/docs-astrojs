> Failed to get the dimensions for `IMAGE_URL`.

## 무엇이 잘못되었나요?

원격 이미지의 크기를 확인하지 못했습니다. 이는 일반적으로 잘못된 URL이나 public 폴더에 있는 이미지 크기를 추론하려고 시도하는 것이 불가능한 경우에 발생합니다.
