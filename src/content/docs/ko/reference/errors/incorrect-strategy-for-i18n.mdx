> **IncorrectStrategyForI18n**: The function `FUNCTION_NAME` can only be used when the `i18n.routing.strategy` is set to `"manual"`.

## 무엇이 잘못되었나요?
일부 국제화 함수는 Astro의 자체 i18n 라우팅이 `i18n.routing: "manual"` 구성 설정에 의해 비활성화된 경우에만 사용할 수 있습니다.
