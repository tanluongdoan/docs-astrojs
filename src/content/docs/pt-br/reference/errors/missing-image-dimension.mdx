> Atributos width e height faltando para `URL_IMAGEM`. Ao usar imagens remotas, ambas as dimensões são necessárias para evitar cumulative layout shift (CLS).

## O que deu errado?

Para imagens remotas, `width` e `height` não podem ser inferidos automaticamente a partir do arquivo original. Para evitar cumulative layout shift (CLS), ou especifique essas duas propriedades ou defina [`inferSize`](/pt-br/guides/images/#infersize) como `true` para baixar as dimensões originais de uma imagem remota.

Se sua imagem está dentro do seu diretório `src`, você provavelmente quis importá-lo no lugar. Veja [o guia de Importações para mais informações](/pt-br/guides/imports/#outros-assets).
