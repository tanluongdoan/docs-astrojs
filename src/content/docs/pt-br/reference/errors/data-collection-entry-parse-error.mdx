> `COLLECTION_ENTRY_NAME` falhou ao ser analisado.
## O que deu errado?
As entradas de coleção do `type: 'data'` devem retornar um objeto com JSON válido (para entradas `.json`) ou YAML (para entradas `.yaml`).
